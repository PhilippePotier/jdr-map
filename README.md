# JDR Map 🗺

Application web progressive (PWA) pour tracker ses déplacements dans un jeu de rôle.

## Installation sur GitHub Pages

1. Crée un repo public sur GitHub (ex: `jdr-map`)
2. Upload tous les fichiers de ce dossier à la racine du repo
3. Va dans **Settings → Pages → Source → Deploy from branch → main → / (root)**
4. Ton app sera accessible sur `https://TON-PSEUDO.github.io/jdr-map`

## Installer comme app sur iPad

1. Ouvre l'URL dans **Safari** sur ton iPad
2. Appuie sur le bouton **Partager** (carré avec flèche)
3. Choisis **"Sur l'écran d'accueil"**
4. L'app apparaît comme une vraie app, fonctionne hors-ligne

## Fonctionnalités

- 🗺 Carte interactive avec déplacement au tap
- 📍 Nommer les cases (appui long)
- 📖 Suivi du chapitre en cours
- ⚡ Stats : Habilité, Endurance, Chance, Or, Équipement, Trésor, Provisions
- ⚔️ Section combat avec dés
- 🎲 Lancer de dés (1 ou 2)
- 💾 Sauvegarde automatique locale
- 🗂 Plusieurs campagnes indépendantes
- 📴 Fonctionne hors-ligne
